# 平台介绍

木瓜 API 是一个面向开发者的统一 AI API 接入站点。你可以通过固定接口地址接入可用模型，并在应用、脚本、服务端程序和 CLI 工具中调用。

## 核心能力

- OpenAI 兼容接口
- API Key 管理
- 模型列表查询
- 聊天补全调用
- Claude Code、Codex、OpenCode、OpenClaw 等 CLI 工具接入

## 入口地址

控制台：

```text
https://api.mugua.link
```

API：

```text
https://api.mugua.link/v1
```
